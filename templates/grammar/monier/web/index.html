The MW Dictionary on the Web

The University of Cologne has created a digital version of the Monier-Williams dictionary, which can be found here. For help with formatting the words you want to search for, you can visit their helpful help page. The help page, however, is not exhaustive; for that reason, this page is here to point out two neat features of the dictionary.

Viewing the original page

To view the original dictionary page, click the link next to the word. This function is useful if you want to see words close to the one you have looked up. Since the digital dictionary is not quite perfect, this feature is useful; you can search for a word that is alphabetically close to the one you are looking for and manually search for it on just a few pages.

Click the link next to the word to load a copy of the dictionary page. Dictionary meanings are not displayed in this image, but they are displayed normally.

Searching for partial words

You can search for pieces of words by using the advanced search option available in the upper right. You can also search for English meanings.

adas

Introduction

The last of the four main pronouns is pronoun is adas. Sanskrit pronouns differ primarily on the basis of the distance between the speaker and the object in question. adas fits on the spectrum as follows:

  • एतद् → इदम् → अदस् → तद्

    etad → idam → adas → tad

    this (right near us) → this (close at hand) → that (within sight) → that (remote)
